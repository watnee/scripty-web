Scripty API — link relations

Responses from /api use HAL. Standard IANA relations such as self keep their usual meaning; the project-specific relations below are namespaced scripty:* via a HAL curie, and each links here by its name. A client should follow these relations rather than hard-coding URLs — presence of a mutation relation also signals that the current user is allowed to perform it. Send Accept: application/prs.hal-forms+json to additionally receive _templates describing each mutation's HTTP method and fields.
