🕒 History & Undo

Available on the project page when the history toolbar is visible.

Undo edit Cmd + Z
Redo edit Cmd + Shift + Z

🔍 Search & Navigation

On the project page:

Search script Cmd + F
Close search Esc
Outline mode Toolbar outline button
Scene outline navigator Toolbar list icon
View keyboard shortcuts ?

↕️ Block Reordering

Focus a block row or hover over it, then use:

Move block up Alt + ↑ or Ctrl + Shift + ↑
Move block down Alt + ↓ or Ctrl + Shift + ↓

Move shortcuts are disabled while typing in a block textarea.

📋 Text Selection

Select text across blocks in the script view, then use:

Copy selection Cmd + C
Cut selection Cmd + X
Paste at caret Cmd + V
Delete selection Delete or Backspace

Paste from Scripty keeps element types. Paste from elsewhere (Fountain or plain screenplay text) splits into typed blocks when it looks like a script; short prose stays in the current block.

✍️ Inline Editing

While editing blocks, scenes, or project titles inline:

Save / submit block Enter
New line in block Shift + Enter
Cycle element type Tab
Cycle element type backward Shift + Tab
Spelling suggestions Cmd + .
Autocomplete navigate ↑ / ↓
Accept suggestion Enter or Tab
Dismiss autocomplete Esc
Delete empty block Backspace or Delete
Cancel empty create row Esc
Save scene name Enter
Cancel project rename Esc

🔤 Text Size

On the project page:

Increase text size Cmd + +
Decrease text size Cmd + −

💡 Tip

While typing in a block, undo and redo change text within that field. Otherwise they undo or redo block create, delete, move, and edit actions for the whole project.
