🕒 History & Undo

Available on project and scene pages when the history menu is visible.

Undo edit Cmd + Z
Redo edit Cmd + Shift + Z

↕️ Block Reordering

Focus a block row or hover over it, then use:

Move block up Alt + ↑ or Ctrl + Shift + ↑
Move block down Alt + ↓ or Ctrl + Shift + ↓

✍️ Inline Editing

While editing blocks, scenes, or project titles inline:

Save / submit block Enter
New line in block Shift + Enter
Save scene name Enter
Cancel project rename Esc

💡 Tip

Block move shortcuts save any open inline edit first, then reorder the row. Undo and redo work on text fields directly; elsewhere they trigger the history menu actions.
