🎬 Welcome to Scripty

Scripty is a web-based collaborative screenwriting platform. It breaks screenplays down into semantic hierarchical units rather than page-based text, allowing for granular editing, character management, versioning, and shared access with automatic refresh while you’re not typing.

📐 The Scripty Hierarchy

Scripts in Scripty are structured using two primary entities:

  • Projects: The top-level screenplay document.
  • Blocks: Granular paragraphs typed after the Fountain screenplay elements — Scene Heading, Action, Text, Character, Dialogue, Dual Dialogue, Parenthetical, Transition, Shot, Lyrics, Centered text, Section, Synopsis, Note, and Page Break. A scene-heading block groups the blocks that follow it; use the Element buttons on the project page to change the type of selected rows. Dual Dialogue marks the second simultaneous speaker (Fountain ^). Text is plain body copy (prints like Action, without screenplay force-markers on Fountain export).

🗑 Deleted a project by mistake?

Delete moves a project to the trash rather than erasing it. It keeps its scenes, characters, version history, editions, songs, and notes, and comes back with all of them intact.

Restoring is an admin job: an admin opens Trash from the Projects toolbar — it appears once something is in there — and clicks Restore. If you are not an admin, ask one to restore it for you. Projects in the trash are deleted for good after 30 days, so don't sit on it.

⭐ Default Project

Click the Set as Default badge at the top of any Project page. When set, that project will load automatically whenever you navigate to the main dashboard of Scripty, bypassing lists.

📱 Install as an App

Install Scripty for a full-screen app icon and easier offline writing.

  • Chrome / Edge (desktop or Android) — open the Help menu (❓) → Install App, or use the browser’s install icon / menu.
  • Safari on iPhone or iPad — tap Share → Add to Home Screen → Add. Or use Help → Install App for a short walkthrough.

After installing, open a project once while online so it can be edited offline later.

💡 Quickstart Workflow Tip

Begin by creating a new Project under Projects — you'll land in the screenplay editor. Click inside the first block to start typing. Use the Scene element button in the toolbar to format a line as a scene heading. To manage characters, go to Casting first to record actors, then add corresponding Characters inside your project's settings.

✍️ Inline Editing & Auto-Save

Simply click on the text area of any block to edit its dialogue or action description inline. Your changes are automatically saved in the background after you stop typing (500ms delay).

📴 Offline Editing

Visit a project once while online so Scripty can cache it. Then you can reopen it offline (from the offline project list, an installed app, or by revisiting the project URL). Edit block text, press Enter to add blocks below, or use the + button. Changes save locally, survive reloads and closed tabs, and sync automatically when you reconnect (with retries if the network is flaky). Unsynced blocks show a small dot indicator. Delete, reorder, and bulk actions still need a connection. See Install as an App for home-screen setup.

➕ Adding Blocks

To insert a block, click the plus icon (+) on the left side of any block, or press Enter while editing. + adds a block below, then opens a menu to choose the element type (or insert a song/note). You can also change type later from Elements in the toolbar, or with Cmd + Option + 1–Cmd + Option + 7.

⌨️ Fountain Power Features

  • Element shortcuts — Cmd + Option + 1–Cmd + Option + 7 set Scene, Action, Character, Parenthetical, Dialogue, Transition, or Shot. Letters cover the rest: T Text, U Dual, Y Lyrics, M Centered, X Section, O Synopsis, N Note, B Page Break (all with Cmd/Ctrl+Option/Alt; works while typing). In Outline mode, 1–3 set Scene, Section, and Synopsis. With the Elements menu open, press the digit or letter directly.
  • Tab / Shift+Tab — move the active block to the next (or previous) logical screenplay element type: Scene → Action → Character → Parenthetical → Dialogue → Transition → Shot.
  • Live syntax — type force markers like .INT. HOUSE, @JANE, >CUT TO:, ~lyrics, or [[note]] and Scripty sets the element type automatically.
  • Character autocomplete — while typing a character cue (or @NAME in Action), pick from your project’s cast list with ↑/↓, then Enter or Tab.
  • Scene autocomplete — while typing a scene heading (or INT./EXT. stubs), pick a standard prefix, reuse a location from prior scenes, pick a full prior heading, or complete time-of-day after - (DAY, NIGHT, DAWN, …).
  • Outline mode — Cmd + Shift + O, or View → Outline mode, filters the script to Scene, Section, and Synopsis blocks only. Enter creates a new Scene; Tab cycles Scene → Section → Synopsis.
  • Page view — Cmd + Shift + P, or View → Page view, lays out the screenplay as US Letter pages with page numbers. Explicit page-break blocks start a new page. Editing still works; pages reflow as you type.
  • Outline / Characters / Locations / Songs — use Lists in the project toolbar, or Cmd + Shift + L for the outline navigator. Outline jumps via Combined (scenes + bookmarks), Scenes, or Bookmarks. Character list and Location list open side panels for the project’s cues and scene locations. Song list jumps to each run of Lyrics blocks in the screenplay.
  • Paste — paste from Scripty keeps element types. Paste Fountain or plain screenplay text from elsewhere and Scripty splits it into typed blocks (scenes, character cues, dialogue, etc.). Short prose without screenplay cues stays in the current block.
  • Import / Export / Print — use File in the project toolbar. Import accepts Fountain (.fountain), plain text, Word (.docx / .doc), Final Draft (.fdx), EPUB (.epub), or PDF (.pdf); Word and PDF files use screenplay layout when present so Scripty-exported DOCX/PDF round-trips element types, otherwise plain text is parsed with Fountain heuristics. Best PDF results come from Scripty-exported or standard screenplay-layout PDFs; scanned or image-only PDFs are not supported (no OCR). Password-protected PDFs must be unlocked first. Final Draft files map Scene Heading, Action, Character, Dialogue, Parenthetical, Transition, Shot, and related elements into typed blocks. EPUB files exported from Scripty round-trip element types exactly; EPUBs from other tools import as plain text parsed with Fountain heuristics. Import replaces all blocks in the project. Export as PDF, Word (.docx) (industry US Letter / Courier layout), Final Draft (.fdx), EPUB (.epub) (a reflowable ebook with one chapter per scene, for e-readers and phones), or Fountain. Fountain round-trips Title:, Credit:, Author:, Draft date: (version), and Contact:, preserves bold/italic/underline, and forces Action/Character markers when needed so re-import keeps element types. Print opens the browser print dialog.

🎵 Songs & Notes

Write outside the screenplay, then insert when ready. Open Songs or Notes from the project toolbar, the project list, the Production hub, Go to songs… / Go to notes… in the + block menu, or with Cmd + Shift + S / Cmd + Shift + D on the project page. Songs are for lyrics and insert as Lyrics blocks. Notes are for scratch text and insert as Note blocks by default. Use Import on either page to load a .txt, .fountain, .docx, .doc, .fdx, .epub, or .pdf file. On the Songs page, Export downloads the whole songbook — or just the songs you have selected — as plain text, PDF, Word (.docx), or EPUB (.epub) (a reflowable ebook with one song per chapter, for e-readers and phones); the song editor's File menu exports the open song in the same formats. Each line becomes its own block on insert. Edits auto-save as you type, and saving updates every place the song or note was already inserted into the screenplay. In the screenplay, open Song list from the lists menu to jump to each run of Lyrics blocks.

While editing a song, Edit in the top bar has Undo, Redo, and Search, just like the screenplay's toolbar. Undo and Redo step through the changes you have made to that song's lines this session; Search filters the lines to those matching your text.

Deleted something by mistake? Delete moves a song or note to the trash rather than erasing it. Open Trash from the Songs or Notes toolbar — it appears once something is in there — and click Restore to put it back with its lyrics and version history intact. Trashed items are deleted for good after 30 days, or right away if you choose Delete forever in the trash.

↕️ Drag & Drop Reordering

Reorder blocks instantly by dragging the double vertical dots (⋮⋮) on the far-left control panel of any block row — they appear when you hover, click, or place the text cursor in the block (including after pressing Enter). Drag the row to the desired position and drop to save the new order. Click the same handle for block options: Elements (change type), bookmark, pin, or delete. Page view mode hides these controls; turn it off from the toolbar to edit.

📌 Bookmarks & Pinned Blocks

Keep track of key story beats or high-priority dialogue. Toggle the star (☆) to bookmark a block, or click the pushpin (📌) to pin a block for fast reference. Open Lists → Outline, then use the Combined or Bookmarks tab to jump to marked blocks.

📦 Bulk Tagging & Deletions

Need to manage multiple blocks at once? Use the checkboxes on the left control panel of blocks (or the "Select All" checkbox) to perform bulk actions: type tags in the action bar to add tags in bulk, or click the trash can (🗑️) to delete multiple selected blocks.

📖 Safari Reader & Read Script

To read your screenplay in Safari Reader (or other browser reading modes), open a project and tap the Reader icon in Safari’s address bar. The project page is structured as an article — script text, scene headings, and character names are reader-visible, while editing controls are excluded. You can also choose Read Script from the View menu for a minimal read-only page that works especially well with Reader.

🔍 Text Size Controls

Improve screenplay readability. Open the View menu and use A− / A+ to scale text size down or up for your display.

Spellcheck & Suggestions

Use Spellcheck in the Tools menu to turn checking on or off. Misspelled words in script blocks are underlined while you edit. Right-click a misspelled word (or press Ctrl+. / ⌘+.) to pick a correction, or choose Ignore to stop flagging that word. Character names and scene locations from your project are treated as valid.

🕒 Undo, Redo & Snapshot History

Use Undo, Redo, and Search from Edit in the project toolbar. Open Snapshot History from the clock icon on a version row to browse saved snapshots, compare what changed, and restore an earlier draft of that screenplay version.

💾 Snapshot History

Block edits, adds, deletes, moves, and tag changes trigger auto-saves (updated in place about every 10 minutes when content changes). Each entry shows what changed since the previous save — block edits with previews, plus scene and character changes. Restoring saves a “Before restore” snapshot first so you can undo a mistaken restore. Older auto-saves are pruned (the newest 30 per screenplay version are kept); named snapshots and “Before restore” entries are kept. Use the version menu to create parallel editable screenplay versions. Click the eye icon next to a version to share it with the team — actors, directors, and other viewers only see that shared version. Click the star to make a version the project default for imports when no version is specified.

👥 Actor Casting

The Casting tab allows you to maintain a directory of real-world actors. You can assign actors to script Characters, making it easy to track casting configurations and contact details.

🎭 Characters & Roles

Create and edit script characters. Each block of dialogue can be associated with a specific character name. Click a character name to view their profile, assigned actor, and full list of dialogue blocks.

🤝 Team Collaboration

Collaborate with others! Open Teams from the user menu to form a project group. Assigning a Project to a Team gives members shared access to view the screenplay. Only users with the Writer role can edit the screenplay.
