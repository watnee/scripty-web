🎬 Welcome to Scripty

Scripty is a web-based collaborative screenwriting platform. It breaks screenplays down into semantic hierarchical units rather than page-based text, allowing for granular editing, character management, versioning, and real-time cooperation.

📐 The Scripty Hierarchy

Scripts in Scripty are structured using two primary entities:

  • Projects: The top-level screenplay document.
  • Blocks: Granular paragraphs typed after the Fountain screenplay elements — Scene Heading, Action, Dialogue, Parenthetical, Transition, Lyrics, Centered text, Section, Synopsis, Note, and Page Break. A scene-heading block groups the blocks that follow it; use the Element buttons on the project page to change the type of selected rows.

⭐ Default Project

Click the Set as Default badge at the top of any Project page. When set, that project will load automatically whenever you navigate to the main dashboard of Scripty, bypassing lists.

💡 Quickstart Workflow Tip

Begin by creating a new Project under Projects, then click inside the first block to start typing. Use Create New Scene to add a scene heading element, or the Scene button in the toolbar to turn selected rows into scene headings. To manage characters, go to Casting first to record actors, then add corresponding Characters inside your project's settings.

✍️ Inline Editing & Auto-Save

Simply click on the text area of any block to edit its dialogue or action description inline. Your changes are automatically saved in the background after you stop typing (500ms delay).

➕ Adding Blocks

To insert a block, click the plus icon (+) on the left side of any block. This inserts a fresh input text area directly underneath, preserving the script's chronological sequence.

↕️ Drag & Drop Reordering

Reorder blocks instantly by dragging the double vertical dots (⋮⋮) on the far-left control panel of any block row. Drag the row to the desired position in the table and drop to save the new order.

📌 Bookmarks & Pinned Blocks

Keep track of key story beats or high-priority dialogue. Toggle the star (☆) to bookmark a block, or click the pushpin (📌) to pin a block for fast reference.

📦 Bulk Tagging & Deletions

Need to manage multiple blocks at once? Use the checkboxes on the left control panel of blocks (or the "Select All" checkbox) to perform bulk actions: type tags in the action bar to add tags in bulk, or click the trash can (🗑️) to delete multiple selected blocks.

📖 Safari Reader & Read Script

To read your screenplay in Safari Reader (or other browser reading modes), open a project and tap the Reader icon in Safari’s address bar. The project page is structured as an article — script text, scene headings, and character names are reader-visible, while editing controls are excluded. You can also use Read Script from the project page, or Read on an individual scene for a minimal read-only view.

🔍 Text Size Controls

Improve screenplay readability. Use the A− and A+ buttons on the top-right of the page to scale text size down or up instantly to suit your display.

🕒 History Dropdown Menu

Access the history menu by clicking the clock icon (🕒) next to the text controls. From here, you can click Undo, Redo, or select Version History to review previous auto-saved stages of the project.

💾 Version History

Block edits, adds, deletes, moves, and tag changes trigger auto-saved versions (at most every 10 minutes when content changes). In Version History, each entry shows what changed since the previous save — block edits with previews, plus scene and character changes.

👥 Actor Casting

The Casting tab allows you to maintain a directory of real-world actors. You can assign actors to script Characters, making it easy to track casting configurations and contact details.

🎭 Characters & Roles

Create and edit script characters. Each block of dialogue can be associated with a specific character name. Click a character name to view their profile, assigned actor, and full list of dialogue blocks.

🤝 Team Collaboration

Collaborate with others! Open Teams from the user menu to form a project group. By assigning a Project to a Team in the project settings, all team members gain shared editing access to work on the screenplay.
